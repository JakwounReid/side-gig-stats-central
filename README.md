# Gig Dash

A dashboard for tracking and analyzing side gig performance metrics — built for people running multiple income streams who want visibility into what's actually working.

**Live:** [side-gig-stats-central.vercel.app](https://side-gig-stats-central.vercel.app)

## What it does

Gig Dash gives side hustlers a single place to track earnings, hours, and performance across gigs — instead of piecing it together from separate apps and spreadsheets.

## Stack

- React · TypeScript · Vite
- Tailwind CSS · shadcn/ui
- Deployed on Vercel

## Running locally

```bash
npm install
npm run dev
```
